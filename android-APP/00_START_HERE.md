## 🚀 **КАК ЗАПУСТИТЬ (3 ПРОСТЫХ ШАГА)**

### **Шаг 1️⃣ - Получите Railway URL**
```
1. Откройте: https://railway.app
2. Перейдите в Dashboard
3. Откройте приложение "Flux"
4. Скопируйте URL (вид: https://flux-production.up.railway.app/)
```

### **Шаг 2️⃣ - Обновите build.gradle**
```
Откройте: android-APP/app/build.gradle
Найдите: buildTypes { release { ... } }
Обновите:
→ buildConfigField "String", "API_BASE_URL", 
   "\"https://flux-production.up.railway.app/\""
```

### **Шаг 3️⃣ - Пересоберите и запустите**
```bash
cd android-APP
./gradlew clean build
./gradlew installDebug
```
