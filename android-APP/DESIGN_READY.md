## 🚀 БЫСТРЫЙ СТАРТ

### Пересобрать и установить:
```bash
cd android-APP
./gradlew clean build
./gradlew installDebug
```

### Или запустить на эмуляторе:
```bash
./gradlew installDebug
adb shell am start -n com.renixst.flux/.ui.activity.LoginActivity
```
